AgriClimate Connect API

Build the future of agriculture. Our API provides real-time access to hyperlocal weather data, AI-driven crop recommendations, soil health analytics, and global market intelligence.

Introduction

The AgriClimate Connect API is organized around REST. Our API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

You can use the AgriClimate Connect API in test mode, which does not affect your live data or interact with the banking networks. The API key you use to authenticate the request determines whether the request is live mode or test mode.

Base URL

https://api.agriclimate.com/v2

Quick Start Guide

This guide will walk you through making your first API call to retrieve weather data for a specific location.

1. Sign up for an API Key

All requests to the AgriClimate API must include a valid API key. You can generate one from your Developer Dashboard.

2. Make your first request

Use the following cURL command to get current weather conditions for a farm in Kenya.

cURL 
curl -X GET "https://api.agriclimate.com/v2/weather/current?lat=-1.2921&lon=36.8219" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

3. Check the response

If successful, you'll receive a JSON payload with detailed weather metrics.

JSON Response 200 OK 
{
  "status": "success",
  "data": {
    "location": {
      "lat": -1.2921,
      "lon": 36.8219,
      "region": "Nairobi"
    },
    "current": {
      "temp_c": 24.5,
      "humidity": 65,
      "precip_mm": 0.0,
      "uv": 8.0,
      "condition": {
        "text": "Partly cloudy",
        "code": 1003
      }
    },
    "agricultural_alerts": [
      {
        "type": "irrigation_advisory",
        "severity": "low",
        "message": "Soil moisture optimal. No irrigation needed today."
      }
    ]
  }
}

Authentication

The AgriClimate Connect API uses API keys to authenticate requests. You can view and manage your API keys in the Dashboard.

Your API keys carry many privileges, so be sure to keep them secure! Do not share your secret API keys in publicly accessible areas such as GitHub, client-side code, and so forth.

Warning: All API requests must be made over HTTPS. Calls made over plain HTTP will fail. API requests without authentication will also fail.

Authentication to the API is performed via HTTP Basic Auth. Provide your API key as the basic auth username value. You do not need to provide a password.

Retrieves a 7-day hyperlocal weather forecast including agricultural-specific metrics like evapotranspiration and soil temperature.
